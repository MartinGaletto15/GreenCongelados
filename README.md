# GreenCongelados ❄️

Bienvenido al repositorio principal de **GreenCongelados**, un sistema integral diseñado para la gestión y comercialización de productos congelados.

Este proyecto utiliza una estructura de monorepositorio para mantener sincronizados el backend y el futuro frontend.

## 🏗️ Estructura del Proyecto

Actualmente, el proyecto se divide en los siguientes módulos:

| Módulo | Descripción | Estado |
| :--- | :--- | :--- |
| [**GC-backend**](./GC-backend/) | API REST desarrollada en .NET 8 con Clean Architecture. | 🚧 **En desarrollo** |
| **GC-frontend** | Aplicación cliente (Web). | 📅 Próximamente |

## 🚀 Estado del Desarrollo

> [!IMPORTANT]
> El sistema se encuentra actualmente en fase de **desarrollo activo**. 
> La lógica central del **Backend** está siendo implementada y refinada.

Para ver detalles técnicos, requisitos de instalación y cómo ejecutar la API, consulta el:
👉 [**README del Backend**](./GC-backend/README.md)

## 🛠️ Tecnologías Principales

- **Backend:** .NET 8, EF Core, SQL Server.
- **Base de Datos:** Diseñada con [Patrones de Dominio](https://en.wikipedia.org/wiki/Domain-driven_design).
- **Documentación:** Swagger / OpenAPI en el backend.
